# Règles du projet

## Git — Auteur des commits

**JE SUIS L'UNIQUE AUTEUR DE CE PROJET.**
- N'ajoute JAMAIS de ligne `Co-authored-by`, `Co-Authored-By` ou toute autre attribution à Claude, Anthropic ou tout autre tiers dans les messages de commit.
- Les commits doivent avoir uniquement mon identité Git (`user.name` et `user.email`).
- Ne modifie aucun fichier de configuration qui pourrait ajouter automatiquement ce type de signature.
