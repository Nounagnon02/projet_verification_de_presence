Pour réinitialiser le mot de passe d'un étudiant, suivez ces étapes institutionnelles :

  1. Naviguez vers le menu Gestion des Étudiants.
  2. Recherchez l'étudiant via son ID matricule (ex: UAC-2024-X).
  3. Cliquez sur les actions secondaires (...) et sélectionnez Générer nouveau mot de passe.
  4. Un lien de réinitialisation temporaire sera envoyé à son adresse académique officielle.
info

Note : L'étudiant doit changer son mot de passe temporaire dès sa première connexion.
